# Weekly Reporting Agent (Code Package)

This package contains all runtime code for the weekly SEO reporting agent.

Main entry points:
- `weekly_seo_agent/weekly_reporting_agent/main.py`
- `weekly_seo_agent/weekly_reporting_agent/workflow.py`
- `weekly_seo_agent/weekly_reporting_agent/reporting.py`
- `weekly_seo_agent/weekly_reporting_agent/clients/*`

Compatibility wrappers remain in `weekly_seo_agent/*` so existing imports/scripts keep working.
